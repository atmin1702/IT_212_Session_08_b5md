# BÀI 5: Thực hành Thiết kế Cổng Kiểm toán đối soát giao dịch (Creative/Advanced)

## 1. Mô tả ngắn gọn ý đồ thiết kế quy trình đối soát giao dịch
Quy trình đối soát giao dịch (Reconciliation Process) giữa hệ thống Core Banking của SecureBank (múi giờ GMT+7) và đối tác thanh toán quốc tế Visa/Mastercard (múi giờ GMT+0 - UTC) được thiết kế theo các nguyên tắc sau:
1. **Lấy mã định danh (Transaction ID) làm khóa đối khớp chính:** So sánh trực tiếp từng bản ghi giữa hai nguồn dữ liệu dựa trên mã giao dịch duy nhất.
2. **Đồng bộ hóa múi giờ trước khi so khớp:** Sử dụng Java 8 Date Time API để phân tích ngày giờ từ Core Banking (GMT+7) và Visa/Mastercard (GMT+0), chuyển đổi cả hai về cùng múi giờ UTC (ZonedDateTime) để đảm bảo so sánh thời gian chính xác tuyệt đối.
3. **Phát hiện các điểm sai lệch (Discrepancies):** Đối chiếu chi tiết số tiền (Amount) và trạng thái giao dịch (Status).
4. **Nhận diện giao dịch mồ côi (Orphan transactions):** Xác định các giao dịch chỉ có ở Core Banking hoặc chỉ có ở Visa/Mastercard.
5. **Ghi nhận và báo cáo:** Gom tất cả lỗi sai lệch vào một danh sách báo cáo chi tiết và ghi nhật ký hệ thống qua `@Slf4j` phục vụ công tác kiểm toán (audit trail).

### [Lượt chat 3] Mã nguồn Java class ReconciliationService hoàn chỉnh
```text
--- AI RESPONSE - BƯỚC 3 ---
[CoT - Phác thảo thuật toán từng bước]
1. Định nghĩa cấu trúc Class:
   - `Transaction`: Biểu diễn giao dịch gồm id, amount, status, stringDateTime, timezone.
   - `Discrepancy`: Chứa thông tin về lỗi lệch (Transaction ID, Tên trường lệch, Giá trị Core, Giá trị Partner).
   - `ReconciliationResult`: Chứa các danh sách kết quả đối soát.
2. Chuẩn hóa thời gian: Viết hàm helper `convertToUtc` nhận vào `Transaction` và trả về `ZonedDateTime` đã chuyển đổi sang UTC. Nếu format lỗi, trả về null để log cảnh báo.
3. Thuật toán đối khớp:
   - Đưa danh sách `partnerTxns` vào một Map `partnerMap` để tìm kiếm theo ID với thời gian O(1).
   - Duyệt qua danh sách `coreTxns`:
     - Nếu ID khớp trong `partnerMap`: So sánh số tiền (sử dụng compareTo của BigDecimal), trạng thái (không phân biệt hoa thường), và thời gian UTC. Nếu có bất kỳ trường nào lệch, tạo đối tượng `Discrepancy`, add vào danh sách và ghi log cảnh báo bằng `@Slf4j`. Nếu khớp hoàn toàn, lưu ID vào danh sách khớp. Xóa giao dịch này khỏi `partnerMap`.
     - Nếu ID không có trong `partnerMap`: Đưa giao dịch vào danh sách chỉ có ở Core (`coreOnlyTxns`).
   - Các giao dịch còn lại trong `partnerMap` sau vòng lặp chính là các giao dịch chỉ có ở Partner (`partnerOnlyTxns`).
4. Ghi log tiến trình và trả về kết quả đối soát.

[Mã nguồn Java class ReconciliationService]

package com.securebank.service;

import lombok.AllArgsConstructor;
import lombok.Data;
import lombok.extern.slf4j.Slf4j;
import org.springframework.stereotype.Service;

import java.math.BigDecimal;
import java.time.LocalDateTime;
import java.time.ZoneId;
import java.time.ZonedDateTime;
import java.util.*;
import java.util.stream.Collectors;

@Slf4j
@Service
public class ReconciliationService {

    @Data
    @AllArgsConstructor
    public static class Transaction {
        private String id;
        private BigDecimal amount;
        private String status;
        private String stringDateTime;
        private String timezone; // Ví dụ: "Asia/Ho_Chi_Minh" hoặc "UTC"
    }

    @Data
    @AllArgsConstructor
    public static class Discrepancy {
        private String transactionId;
        private String fieldName; // "AMOUNT", "STATUS", "DATE"
        private String coreValue;
        private String partnerValue;
    }

    @Data
    public static class ReconciliationResult {
        private List<String> matchedTxnIds = new ArrayList<>();
        private List<Discrepancy> discrepancies = new ArrayList<>();
        private List<Transaction> coreOnlyTxns = new ArrayList<>();
        private List<Transaction> partnerOnlyTxns = new ArrayList<>();
    }

    private ZonedDateTime convertToUtc(Transaction txn) {
        try {
            ZoneId sourceZone = ZoneId.of(txn.getTimezone());
            LocalDateTime ldt = LocalDateTime.parse(txn.getStringDateTime());
            ZonedDateTime sourceZdt = ldt.atZone(sourceZone);
            return sourceZdt.withZoneSameInstant(ZoneId.of("UTC"));
        } catch (Exception e) {
            log.error("Failed to parse/normalize date for txn {}: {}. Error: {}", 
                      txn.getId(), txn.getStringDateTime(), e.getMessage());
            return null;
        }
    }

    public ReconciliationResult reconcile(List<Transaction> coreTxns, List<Transaction> partnerTxns) {
        log.info("Initiating reconciliation. Core Transactions: {}, Partner Transactions: {}", 
                 coreTxns.size(), partnerTxns.size());

        ReconciliationResult result = new ReconciliationResult();

        if (coreTxns.isEmpty() && partnerTxns.isEmpty()) {
            log.info("No data available for reconciliation.");
            return result;
        }

        // Tạo map từ danh sách đối tác để tìm kiếm O(1)
        Map<String, Transaction> partnerMap = partnerTxns.stream()
                .collect(Collectors.toMap(Transaction::getId, txn -> txn, (t1, t2) -> t1));

        for (Transaction coreTxn : coreTxns) {
            String txnId = coreTxn.getId();

            if (partnerMap.containsKey(txnId)) {
                Transaction partnerTxn = partnerMap.get(txnId);
                boolean isMatched = true;

                // 1. Đối soát số tiền
                if (coreTxn.getAmount().compareTo(partnerTxn.getAmount()) != 0) {
                    result.getDiscrepancies().add(new Discrepancy(
                            txnId, "AMOUNT", 
                            coreTxn.getAmount().toString(), 
                            partnerTxn.getAmount().toString()
                    ));
                    log.warn("AMOUNT DISCREPANCY -> ID: {} | Core: {} vs Partner: {}", 
                             txnId, coreTxn.getAmount(), partnerTxn.getAmount());
                    isMatched = false;
                }

                // 2. Đối soát trạng thái
                if (!coreTxn.getStatus().equalsIgnoreCase(partnerTxn.getStatus())) {
                    result.getDiscrepancies().add(new Discrepancy(
                            txnId, "STATUS", 
                            coreTxn.getStatus(), 
                            partnerTxn.getStatus()
                    ));
                    log.warn("STATUS DISCREPANCY -> ID: {} | Core: {} vs Partner: {}", 
                             txnId, coreTxn.getStatus(), partnerTxn.getStatus());
                    isMatched = false;
                }

                // 3. Đối soát ngày giờ (sau khi đưa về UTC)
                ZonedDateTime coreUtc = convertToUtc(coreTxn);
                ZonedDateTime partnerUtc = convertToUtc(partnerTxn);

                if (coreUtc == null || partnerUtc == null || !coreUtc.isEqual(partnerUtc)) {
                    result.getDiscrepancies().add(new Discrepancy(
                            txnId, "DATE",
                            coreUtc != null ? coreUtc.toString() : "INVALID",
                            partnerUtc != null ? partnerUtc.toString() : "INVALID"
                    ));
                    log.warn("DATE DISCREPANCY -> ID: {} | Core UTC: {} vs Partner UTC: {}", 
                             txnId, coreUtc, partnerUtc);
                    isMatched = false;
                }

                if (isMatched) {
                    result.getMatchedTxnIds().add(txnId);
                }

                partnerMap.remove(txnId); // Loại bỏ khỏi map
            } else {
                // Giao dịch chỉ có ở Core Banking
                result.getCoreOnlyTxns().add(coreTxn);
                log.info("CORE-ONLY TRANSACTION -> ID: {}", txnId);
            }
        }

        // Các giao dịch còn lại trong map chỉ có ở Partner
        result.setPartnerOnlyTxns(new ArrayList<>(partnerMap.values()));
        for (Transaction partnerOnly : result.getPartnerOnlyTxns()) {
            log.info("PARTNER-ONLY TRANSACTION -> ID: {}", partnerOnly.getId());
        }

        log.info("Reconciliation finished. Matched: {}, Discrepancies: {}, Core Only: {}, Partner Only: {}", 
                 result.getMatchedTxnIds().size(), result.getDiscrepancies().size(), 
                 result.getCoreOnlyTxns().size(), result.getPartnerOnlyTxns().size());

        return result;
    }
}
```
